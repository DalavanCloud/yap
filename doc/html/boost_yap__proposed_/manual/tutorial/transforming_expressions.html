Transforming Expressions

Boost.YAP expressions are evaluated explicitly and implicitly:

Explicit transformations are done using the transform() function. See Calc3 for an example. A Boost.YAP transformation is free-form. It can transform an expression into anything -- a new expression of any kind, or even a non-expression value (effectively evaluating the expression)

Implicit transformations are estabished by overriding the customization points provided by Boost.YAP; this is explained in detail later in the tutorial. Implicit transformations are only executed when an expression is evaluated. See Mixed for an example; it overrides the eval_call() customization point.
